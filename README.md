Car Damage Detection Project (YOLOv11)
Project Description
This is an Artificial Intelligence project designed to detect and identify various types of car damages (dents, scratches, and collisions) from images. The project aims to help insurance companies and car repair shops automate the damage assessment process quickly and accurately.

Achievements
High Accuracy: The model reached a success rate of 91.6% (mAP50), which means it can identify even small damages with high precision.

Large Dataset: Trained on a huge collection of over 13,000 images to ensure it works in different lighting and angles.

Modern Technology: Built using the latest YOLOv11 technology, making it fast enough to work on mobile apps in the future.
